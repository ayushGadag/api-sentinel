# API Sentinel

A beginner-friendly Python CLI project for API testing and QA automation practice.

API Sentinel sends API requests, validates responses, and displays clean colored terminal output using Python and Rich.

This project is being built step-by-step to learn:

- Python automation
- API testing
- QA engineering practices
- CLI application development
- Git & GitHub workflow

---

# Features

## Current Features

- Send API requests
- Validate status codes
- Colored CLI output
- Simple and clean structure
- Beginner-friendly architecture

# Installation

## 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/api-sentinel.git
```

---

## 2. Open Project

```bash
cd api-sentinel
```

---

## 3. Create Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Mac/Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 4. Install Dependencies

```bash
pip install -r requirements.txt
```

---

# Run Project

```bash
python main.py
```
